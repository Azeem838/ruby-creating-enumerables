# Enumerable Methods

> This project contains various methods that will be available in the Enumerable module.

The project contains the following methods:

1. #my_each
2. #my_each_with_index
3. #my_select
4. #my_all?
5. #my_any?
6. #my_none?
7. #my_count
8. #my_map - accepts procs and blocks
9. #my_inject
10. #multiply_els
